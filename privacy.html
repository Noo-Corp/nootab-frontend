Noo Tab Privacy Policy

Effective Date: June 30, 2025

Updated on: July 18, 2025

Overview
Noo Tab does not collect or store personal data on any server. All application data is stored locally in the user’s browser using local storage and cookies.

Google User Data Access
If a user connects a Google account, Noo Tab requests the following read-only scopes via Google OAuth:

  • https://www.googleapis.com/auth/calendar.readonly – access to calendar event metadata (e.g., title, time, location)
  • https://www.googleapis.com/auth/gmail.readonly – access to unread email metadata and full message bodies
This access is used solely to display the user’s Google Calendar and Gmail content within the Noo Tab interface. Noo Tab does not send, transmit, or store this data externally.

Use of Google User Data
Accessed data is used only to render calendar events and Gmail content in the browser UI. No data is analyzed, processed, or shared for any other purpose. Noo Tab does not modify, create, or delete any Google user data.

Data Protection
All Google user data is accessed securely via OAuth 2.0. Credentials are never stored or accessible to Noo Tab. Access tokens are held only in the user's browser environment and may be stored in cookies to maintain session state. These cookies do not contain passwords and can be cleared at any time via browser settings or the application's Clear Data function. All data transmission occurs over HTTPS.

Data Retention and Deletion
Google user data is stored only in the user’s browser memory or local storage during use. No data is retained on any server. Users can remove all locally stored data by:

  • Using the Clear Data button in the application
  • Manually clearing browser storage and cookies
  • Revoking Noo Tab’s access from their Google Account settings
Noo Tab performs no persistent logging, caching, or backup of Google user data.

Analytics
Google Analytics is used to collect anonymized usage data such as page views, browser type, and approximate location. This data is not linked to any Google account or personal identity and is used solely to improve the product. Users may opt out via browser settings or Google’s opt-out tool.

Data Sharing
Noo Tab does not share, sell, or transmit any user or Google data to third parties. Google user data is accessed and processed locally only.

User Controls
Users can disconnect their Google account at any time via their Google Account permissions. All locally stored data can be removed through the browser or by using the Clear Data option within the application.
